# 📈 VolumePulse AI

## Smart Stock Volume Spike Detection System

VolumePulse AI is an interactive stock market analytics dashboard that analyzes trading volume trends and detects unusual market activity using Machine Learning and statistical outlier detection techniques.

The project uses live stock market data and provides real-time visualization of:

* Volume trends
* Volume spikes
* Anomaly detection
* Trading insights
* Interactive stock charts

✅ Live Stock Market Data

✅ Outlier Detection using AI

✅ Real-Time Visualization

✅ Beautiful Financial Dashboard UI

---

# 📊 Concepts Used

## 📈 Volume Trend

The system calculates moving averages of stock trading volume to identify increasing or decreasing volume patterns.

## 🚨 Outlier Detection

The project uses:

* Z-Score Statistical Analysis
* Isolation Forest Machine Learning Algorithm

to identify unusual trading activity and abnormal market behavior.

## 📉 Visualization

Interactive charts are created using Plotly to visualize:

* Stock price movement
* Trading volume
* Volume trends
* Detected anomalies
